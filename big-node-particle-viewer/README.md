# 独立模型粒子查看器

这是一个和 `灵境 .html` 不绑定的独立查看器。它只负责一件事：把你选择的 GLB / GLTF 模型转换成粒子点云，并用当前提取出的 shader 逻辑查看效果。

## 打开方式

用 VS Code Live Server 或其他本地服务器打开项目文件夹，然后访问：

```text
http://127.0.0.1:5500/big-node-particle-viewer/index.html
```

不要直接用 `file://` 打开。Three.js module、GLB 加载、贴图读取都更适合在本地服务器里运行。

## 加载模型

你有三种方式：

- 点击“本地”选择一个 `.glb` 文件。
- 把 `.glb` 文件直接拖到面板里的虚线区域。
- 在“路径”里输入相对路径，例如 `./models/example.glb` 或 `../models/example.glb`，再点“加载路径”。

最推荐使用 `.glb`。如果是 `.gltf` 并且依赖外部 `.bin` 或贴图，浏览器需要能按相对路径访问那些资源。

## 它和原项目的关系

它不会修改、引用或覆盖：

```text
/Users/zzziccgeo/Desktop/节点细化/灵境 .html
```

它也不再内置入口天桥、石之寄语、洞之奥义等节点配置。现在它是一个通用工具，任何模型都可以单独加载查看。

## 参数说明

- `预算`：最多采样多少模型顶点。数值越高越细，但越吃性能。
- `粒径`：基础粒子大小。
- `透明`：整体点云透明度。
- `墨化`：原贴图颜色和灰度水墨色的混合比例。
- `自动取景`：根据当前点云包围球自动调整相机。
- `复制参数`：复制当前模型、粒子参数和相机位置，方便记录。

## 文件说明

- `index.html`：独立查看器主体。
- `particle-logic.md`：模型转粒子的实现逻辑说明。
